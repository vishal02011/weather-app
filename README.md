# WeatherNow 🌤️

A responsive weather web app built with React that shows real-time weather data powered by the Open-Meteo API.

## 🛠️ Tech Stack
- React (Vite)
- OpenWeatherMap API
- Recharts (temperature graph)
- CSS Flexbox & Grid

## 📱 Features
- 🔍 Search weather by city
- 🌡️ Current temperature, feels like, weather condition
- 💧 Humidity, wind speed, pressure, visibility, cloud cover
- ⏱️ Hourly forecast with weather icons
- 📅 5-Day forecast
- 📈 Temperature trend graph
- 📱 Fully responsive design
- Animated weather icons

## 🚀 Getting Started

```bash
git clone https://github.com/vishal02011/weatherNow
cd weatherNow
npm install
npm run dev
```
